Scraping data

Getting list of categories and sub-categories

This code is using several Python libraries such as os, glob, pandas, re, and numpy. The purpose of this script is to do some cleaning and to extract phone numbers from the "job description" column of a pandas dataframe that contains job listings and insert them into a new column.

It starts by creating a pandas dataframe by concatenating all job listings that are stored in CSV files in a specific directory. Then it filters out any duplicates and unnecessary spaces in the date field.

The code then uses a regular expression to search for phone numbers in the job description column and insert them into a new column. The resulting dataframe is then saved to a new CSV file in a different directory.

The main benefit of this script is that it allows the user to extract phone numbers from job listings, which can be useful for various purposes. It also provides a simple way to extract data from multiple CSV files and store them in a single dataframe for further processing.
